# Предсказатель трафика в метро
## Версия python 3.12
Для работы приложения необходим python 3.12. Пожалуйста, не используйте другие версии!
## Быстрый старт
Активировать виртуальное окружение
```python
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Установить зависимости (в активном venv)
```python
pip install -r requirements.txt
```

Запустить веб-интерфейс (в активном venv)
```python
streamlit run src/main.py
```
